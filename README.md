# Real-Time Weather Dashboard
A responsive Real-Time Weather Dashboard built using HTML, CSS, and JavaScript. It fetches live weather data from a public REST API using Fetch API and async/await, allowing users to search weather information by city name with proper error handling.

## Features
- Fetch real-time weather data using Fetch API
- Async/Await implementation
- Search weather by city name
- Parse nested JSON data
- Comprehensive error handling
- Dynamic UI rendering

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- REST API
- Fetch API

## Expected Outcome
Displays accurate live weather metrics including:
- Temperature
- Humidity
- Wind Speed
- Weather Condition

wttr.in Public Weather API
